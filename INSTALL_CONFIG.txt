
ICI JE VOUS EXPLIQUE COMMENT INSTALLER LES LIBRARY QUE NOUS AVONS BESOIN POUR CE PROJET

VERIFIEZ QUE VOUS AVEC UN DOSSIER cmake_modules AVEC FindSFML.cmake et FindThor.cmake

----- LA PREMIÈRE EST LA SFML

1 - SUIVRE LE TUTO DE LA DOC < OU > BREW INSTALL SFML

----- LA DEUXIÈME EST THOR (UNE EXTENSION DE LA SFML AVEC DES COMPOSENTS BEAUCOUP PLUS PUISSANTS

1 - TÉLÉCHARGER LA SOURCE (version 2.0) DE THOR
    COPIER COLLER CE LIEN DANS VOTRE NAVIGATEUR
    --> http://www.bromeon.ch/libraries/thor/download/v2.0/thor-v2.0-sdk.7z (POUR LE DÉZIPER EXTRACTOR SUR MAC FAIT LA FAIRE)

2 - TÉLÉCHARGER CMAKE POUR MAC OS X --> https://cmake.org/download/
    2.1 - ALLEZ DANS LE DOSSIER "INCLUDE" ET COPIER COLLER LE DOSSIER "Thor" DANS "/usr/local/include"
    2.2 - ALLEZ DANS LE DOSSIER "EXTLIBS/AURORA" ET COPIER COLLER LE DOSSIER "Aurora" DANS "/usr/local/include"

3 - OUVREZ CMAKE ET AVEC L'AIDE DE CE TUTO (http://www.bromeon.ch/libraries/thor/tutorials/v2.1/installation.html)
    BUILD THOR
    3.1 - DANS "where is the source code" METTEZ LA RACINE DU DOSSIER QUE VOUS AVEZ TÉLÉCHARGÉ ET DÉZIPÉ
        (EX = /Users/kevinhuron/Downloads/thor-v2.0-sdk)
    3.2 - DANS "where to build the binairies" METTEZ L'ENDROIT OU VOUS VOULEZ BUILD
        (EX = /Users/kevinhuron/Downloads/thor-v2.0-sdk/build)
    3.4 - CLIQUEZ SUR CONFIGURE
    3.5 - IF "Configuriong Done" CLIQUEZ SUR GENERATE (SI ERROR LAISSER N'Y FAITE PAS ATTENTION POUR LE MOMENT)
    3.6 - ALLEZ DANS LE DOSSIER "BUILD" QUE VOUS AVEZ INDIQUEZ COMME DESTINATION EN 3.2
    3.7 - DANS LE DOSSIER BUILD ALLEZ DANS "SRC" ET COPIER COLLER LE FICHIER "libthor.dylib" DANS "/usr/local/lib"

4 - REVENEZ ICI FAITE CLIQUE DROIT SUR LE PROJET RACINE ET CLIQUEZ SUR "RELOAD CMAKE PROJECT"
    SI PAS D'ERROR --> COOL --> GO LAUNCH THE PROJECT
    SI ERROR --> PAS COOL --> VOIR AVEC KEVIN (LE DIEU)
